Welcome to FlowDeck for Teams!

Your team licenses are ready. Here's how to get everyone set up.

Getting Started
  1. Share Your License Key
    You'll find your license key in the order confirmation email. Share it with your team members — each device activated with the key will count as one seat.
  2. Install FlowDeck CLI
    Each team member runs this command in their terminal:
    curl -sSL https://flowdeck.studio/install.sh | sh
  3. Activate & Start Building
    Run interactive mode to activate with the shared license key:
    flowdeck -i
  4. Read the Docs
    Learn all the features and commands at docs.flowdeck.studio
Managing Your Licenses

Each device activated with your license key counts against your seat quota. You can view and manage active devices from your orders page — deactivate a device anytime to free up a seat.

Need more seats? You can upgrade your plan at any time from the same page.
